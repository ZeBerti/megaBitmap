# MegaBitmap

Demonstrates a simple structure of Information to create a pixelart style Bitmap (Image) by a simple syntax: ([\d+][a-zA-Z])*

Example:  
You like to draw 3 black pixels and 2 white pixels:  
"3s2w "  
■■■□□

"3s 2w 2s 3w "  
■■■□□■■□□□
